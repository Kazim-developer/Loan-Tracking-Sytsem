# Receivable & Installment Management SaaS

This SaaS helps lenders and businesses track loans, invoices, installments, payments, and send reminders via WhatsApp and email.

## Tech Stack

- Next.js
- Node.js
- Prisma
- PostgreSQL
- Tailwind

## Modules

- Clients
- Accounts (Loans & Invoices)
- Installments
- Payments
- Reminders
- Dashboard
